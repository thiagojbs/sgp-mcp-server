ü§ñ Complete MCP Integration Guide

Step 1: Understanding MCP Protocol

The Model Context Protocol (MCP) is a JSON-RPC 2.0 based protocol that allows automated systems to interact with external services. This SGP server implements MCP via HTTP/SSE for maximum compatibility.

Key MCP Methods:
  • initialize - Start MCP session
  • tools/list - Get available tools
  • tools/call - Execute a tool
  • resources/list - Get available resources
  • prompts/list - Get available prompts

Step 2: Multi-Tenant Configuration

Each client can use their own SGP credentials by passing them in HTTP headers. This ensures complete data isolation.

Required Headers:
X-SGP-URL: https://your-sgp-server.com/api
X-SGP-APP: your-app-name  
X-SGP-TOKEN: your-api-token

Step 3: Using SGP Tools

The server provides 9 tools for interacting with SGP data. Some tools require customer credentials (CPF/CNPJ + password).

Functional Tools (100% Working):
  • sgp_test_connection - Test API connectivity
  • sgp_get_customer_contracts - Get customer contracts
  • sgp_get_customer_invoices - Get customer invoices
  • sgp_get_network_status - Get network status

Step 4: Error Handling

MCP clients should be prepared to handle various error scenarios and implement appropriate fallback strategies.

Common Error Patterns:
  • 404 errors for unsupported endpoints
  • Authentication failures with invalid credentials
  • Network timeouts for slow responses
  • Rate limiting on heavy usage

Integration Examples 

import asyncio
import aiohttp
import json

class SGPMCPClient:
    def __init__(self, sgp_url, sgp_app, sgp_token):
        self.base_url = "https://sgp-mcp-server.thiagojbs.workers.dev"
        self.headers = {
            'Content-Type': 'application/json',
            'X-SGP-URL': sgp_url,
            'X-SGP-APP': sgp_app,
            'X-SGP-TOKEN': sgp_token
        }
    
    async def call_tool(self, tool_name, arguments={}):
        payload = {
            "jsonrpc": "2.0",
            "id": 1,
            "method": "tools/call",
            "params": {"name": tool_name, "arguments": arguments}
        }
        
        async with aiohttp.ClientSession() as session:
            async with session.post(f"{self.base_url}/mcp", 
                                   headers=self.headers, 
                                   json=payload) as response:
                return await response.json()

class SGPMCPClient {
  constructor(sgpUrl, sgpApp, sgpToken) {
    this.baseUrl = 'https://sgp-mcp-server.thiagojbs.workers.dev';
    this.headers = {
      'Content-Type': 'application/json',
      'X-SGP-URL': sgpUrl,
      'X-SGP-APP': sgpApp,
      'X-SGP-TOKEN': sgpToken
    };
  }

  async callTool(toolName, arguments = {}) {
    const response = await fetch(`${this.baseUrl}/mcp`, {
      method: 'POST',
      headers: this.headers,
      body: JSON.stringify({
        jsonrpc: '2.0',
        id: Date.now(),
        method: 'tools/call',
        params: { name: toolName, arguments }
      })
    });
    return await response.json();
  }
}
